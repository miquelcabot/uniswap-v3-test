# uniswap-v3-test
Test app that connects to Uniswap v3 using Hard Hat + Waffle + Ethers

Install:
```
yarn install
```

Compile:
```
yarn run compile
```

Test in local Hardhat blockchain:
```
yarn run test
```

Test in Rinkeby test network:
```
yarn run test_rinkeby
```

Run prettier on all your contracts:
```
yarn run prettier
```
